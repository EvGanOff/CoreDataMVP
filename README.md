# CoreDataMVP
## Простое приложение состоящее из двух экранов показывающее работу CoreData. 
### Стек: UIKit, MVP, CoreData, Coordinator, DI.
#### Простое приложение состоящее из двух экранов показывающее работу CoreData.
 
На первом экране расположены UI элементы (UITextField, UIButton, UITableView). 
При нажатии на кнопку, текст из UITextField попадает в хранилище, при этом данные в таблице обновляются. 
Реализована обработка свайпа по ячейке, при котором ячейка и данные удаляются.

Второй экран с профилем. По умолчанию все элементы не активны. Кнопка Edit активирует режим редактирования. 
После редактирования кнопка Save сохраняет данные в хранилище.
